# 输入输出目录规范

## 输入（固定在 `data/`）

- 事件输入：`data/test_events.json`
- 真实基准：`data/benchmark_events_real.json`
- 模拟基准：`data/benchmark_synthetic_cases.json`
- 分类案例库：`data/case_catalog/`

## 中间数据（缓存）

- Step1 事件缓存：`data/events/<event_id>/`
- 真实实验缓存：`data/experiments/real_events/<event_id>/`

## 输出（固定在 `report/`）

- 溯源过程报告：`report/forensics/forensics_*.json`
- 评估统计报告：`report/evaluation/*.json`

## 统一路径常量

路径常量统一定义在：`tools/project_paths.py`

后续新增脚本请直接引用该模块，不再硬编码 `data/...` 或 `report/...`。
